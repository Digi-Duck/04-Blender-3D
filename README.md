## 專案介紹

當你想到中興大學綜合大樓-八樓上課洽公卻怕迷失方向?

或是你想看看目前八樓有什麼超酷超炫的課?

`布蘭德．珊滴`讓你先探索這個未知的空間。

### 使用技術

HTML / CSS / SCSS / Three.js
